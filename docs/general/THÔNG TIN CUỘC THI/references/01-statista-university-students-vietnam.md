# [1] Number of University Students in Vietnam (2013–2021)

**Source:** Statista
**URL:** https://www.statista.com/statistics/815091/number-of-university-students-in-vietnam/
**Accessed:** February 2024

## Data

| Year | Students (millions) |
|------|-------------------|
| 2013 | 1.67 |
| 2014 | 1.82 |
| 2015 | 1.75 |
| 2016 | 1.77 |
| 2017 | 1.71 |
| 2018 | 1.53 |
| 2019 | 1.67 |
| 2020 | 1.91 |
| 2021 | 2.15 |

## Key Finding

In 2021, there were approximately 2.15 million university students in Vietnam, indicating an increase from the previous year. Vietnam had reached 242 universities by 2021.

Growth from 1.67 million (2013) to 2.15 million (2021) = 29% increase over the period.
